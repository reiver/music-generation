Music Generation

the creation of musical material using automated or semi-automated systems that specify, selects, or organizes musical events according to some procedure or algorithm

by Charles Iliya Krempeaux

Music Generation is the process of creating new music using a set of instructions, rules, patterns, or algorithms instead of writing every note by hand.

The system doing the generating might be a person following musical rules, a mechanical device, a computer program, or an AI model.

That process could involve rolling dice, following the rules of a musical tradition, using a mathematical formula, running a computer algorithm, asking an AI to create a song, etc.

The general idea is that a system — whether carried out by a human, a mechanical device, a computer, or an AI — helps produce the musical result according to some set of instructions, rules, patterns, or algorithms
